Accessible use

  • Make sure button labels accurately describe the action the button carries out.
  • If the button's visible label is an icon or other image, include a screen-reader-friendly text version.
  • If the button is hidden, ensure it is also hidden on screen readers.
  • Buttons should be at least 44x44px in size.
